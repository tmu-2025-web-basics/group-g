駐日韓国文化院（주일한국문화원）

四谷三丁目

Korea/韓国

四谷三丁目で味わうプチ韓国旅行＿＿＿

入口の柱には寺院や宮殿などの木造建築に使われれう装飾「丹青（タンチョン）」が再現されている。
두근두근（トゥグントゥグン＝ドキドキ）の入場の瞬間！！

「하늘정원」（ハヌルジョンウォン）：ハヌル庭園
「ハヌル」とは「空」という意味。この軒下で休憩をし、韓国生活を想像しながら、日々の疲れを癒すことができる。どなたでも利用が可能だ。

「ハヌル庭園」ここでは、韓国の昔ながらの懐かしい風情を感じることができる＿＿＿

白い粒砂のような小さな石が一面に広がっていて、青い空とのコントラストがとても映え、美しかった。一度入場すると韓国時代劇にトリップしたような思いに＿

醤油・味噌などを入れて保存する甕（かめ）が庭に設置されている。
名称は「ハンアリ壺」（항아리）。実際に使われていたものだと思うととても感慨深い！

「사랑방」（サランバン）：舍廊房
サランバンに入るときは靴の向きに注意！日本とは逆向きに靴を置く。
私は訪問後日その事実を知り教養へと昇華させることで、「向きを間違えた」という経験による羞恥を回避。

ここでひとつ休憩…！韓国時代劇に思いを馳せて…！ 多くの朝鮮時代の伝統品が展示されているこのサランバン、いちデザイン系学科の私にとってもとても興味深く学びとなった。 「花郎（ファラン）」（韓国時代劇ドラマ）見たいなぁ…

「サランバン」（舍廊房）韓国時代劇でもよく見る伝統家屋 書斎や応接間として使われていた「交流の場」＿＿＿

サランバンは男性の空間を意味する。 奥にある屏風と「ポリョ」と呼ばれる長座布団、背もたれ、長枕、四方枕一式の圧倒的存在感。 屏風は漢字とハングルで構成されているため、解読を試みるも、あえなく惨敗＿

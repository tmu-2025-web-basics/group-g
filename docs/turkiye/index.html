東京ジャーミイ

代々木上原

Turkey/トルコ

イスラム教徒が礼拝する場所である「モスク」の天井部分。
ここでは、イスラム教徒が実際に礼拝する姿を後ろから見学することができる。
礼拝の時間は、信者たちにとって日常の一部であると同時に、 心を整える大切な時間であることがひしひしと伝わってきた。

「集いの場」、「社交の場」としてのトルコ文化センター＿＿＿

ガイドさんのお話によると、モスクの最終的な理想形としては「複合社会福祉施設」とのこと。
トルコのイスタンブールにあるモスク「スレイマニエジャーミイ」では、無料の宿泊施設や病院、飲食を提供する場所があるとのこと。

日本とトルコの国旗。
ガイドさんのお話では、モスクは「人と人が出会う場所」であるという。
日本とトルコをつなぐこのモスクでは様々な人々が会話をし、交流をしている。

トルコの味ぜひ堪能してみてください＿＿

ここトルコ文化センターでは、イスラム教徒でも非イスラム教徒でも訪れることができる。
トルコのお茶やデーツなど自由にいただける。

トルコ文化センターの隣のお土産屋さんでは異国の食べ物などが多く売られている。
ゴマのせんべいを買った班員による感想は、「以外に甘くておいしかった！」とのこと。

SAMARKAND TERRACE

高田馬場

Uzbekistan/ウズベキスタン

高田馬場駅から近く、控えめな照明に中央アジア感じる音楽が良い雰囲気。

プロフはウズベキスタンの国民食で、テーブル席からも見える大鍋で作られる。 ラム肉とニンジンのガツンと感に、ヒヨコ豆とパラパラなお米の相性がよく、 食欲をそそり、さっぱりしたセットのサラダと交互に食べていけば永久機関が完成。
